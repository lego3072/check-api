CheckAPI Integration Docs

MCP-first compliance validation for agent outputs and document workflows.

1) Create a free API key

curl -X POST {{BASE_URL}}/api/signup \\
  -H "Content-Type: application/json" \\
  -d '{"email":"you@company.com"}'

2) Run a compliance check

curl -X POST {{BASE_URL}}/v1/check \\
  -H "Authorization: Bearer YOUR_API_KEY" \\
  -H "Content-Type: application/json" \\
  -d '{"text":"Your generated output", "regulations":["gdpr","soc2"]}'

3) Use MCP endpoints

GET {{BASE_URL}}/v1/mcp/tools
POST {{BASE_URL}}/mcp

Use tool name check_compliance as a required pre-response validation step.
